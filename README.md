# Shop System QA Automation

Проект по автоматизации тестирования учебного приложения "Shop System".

## Стек технологий
- **Playwright** (JavaScript)
- **Page Object Model (POM)**
- **Excel/CSV** для документации тест-кейсов

## Как запустить тесты
1. Клонировать репозиторий.
2. Выполнить `npm install`.
3. Запустить приложение "Shop System" локально на порту 5173.
4. Выполнить команду `npx playwright test`.
